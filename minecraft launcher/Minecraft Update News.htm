Titan Minecraft Launcher News

The launcher version 3.8.2 is now compatible with Minecraft 1.17 and all earlier versions. Later versions may also work. It now adds MythicMC into your server list. We are not directly associated with MythicMC. MythicMC discord here.

Automatic updates from Titan launcher 3.8.0 is sadly broken. You can manually update it from here.
Instructions: Do properties on your current launcher icon. Click "open file location". Save the downloaded new launcher into the same folder.

Updates on this page can be sporadic. Go to minecraft.net or Minecraft Wiki for further info.
